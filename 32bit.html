How to check the CPU Architecture

Following the news: Pokémon GO to discontinue support on 32-bit Android devices.

I've written a brief step-by-step tutorial on how to find out if your phone is going to be affected.
  • Start by downloading and installing the following app : antutu-benchmark
  • Open it and click My Device under Basic Info it will say Android followed by the version and architecture installed.
    If it's says 64-bit you won't be affected and can stop reading this.

    If it says 32-bit swipe down until you see CPU and check the Type if it says 32-bit you won't be able to play pokemon go with that phone. If it says 64-bit you will have to install a 64-bit custom rom.
